##### 변수
- 변수 : 값을 담는 상자. 값을 재사용하기 위해 주로 사용
- 변수 타입: 상자에 어떠한 유형의 값을 담을지 결정
- 타입 명시
`[타입] [변수명] = [담음값];`
`int age = 20;`
- 타입 추론 : 타입을 적지 않고 뒤에 오는 값에 따라 타입이 자동으로 저장됨
`var [변수명] = [담음값]; // 변경 가능`
`final [변수명] = [담음값]; // 변경 불가능` 
`const [변수명] = [담음값]; // 변경 불가능 -> 리터럴한 값. 컴파일 시점에 값이 결정되는 변수`

##### 데이터 타입
- 상자에 어떠한 유형의 값을 담을지 결정
 이 4가지가 제일 많이 쓰임
- `int` : 정수
- `double` : 실수(소수점 포함)
- `String` : 문자열
- `bool` : 참/거짓
- `dynamic` : 모든걸 담을 수 있는 타입. 오류발생 가능성 높음!

##### 연산자
- `==` 같은지 비교
- `<` 우측이 큰지 비교
- `>` 좌측이 큰지 비교
- `<=` 우측이 크거나 같은지 비교
- `>=` 좌측이 크거나 같은지 비교
- `*` 산술연산자. 곱하기 
- `-` 산술연산자. 나누기
- `%` 산술연산자. 나머지
- `~/` 산술연산자. 몫
- `산술연산자= b` 산술연산자 뒤에 =을 붙이면. a와 b 연산한 결과를 a에 저장
- `a = b ? c : d` 3항연산자 => a의 값은 b가 참이면 c, 거짓이면 d!
- `a = b ?? c` null 병합연산자 => a의 값은 b가 널이면 c, 아니면 b!

##### 조건문
- `if(조건식) { 실행할코드 }`
- `if(조건식1) { } else if(조건식2) {} else {}` 조건식 1이 참이 아닐 경우, 조건식2 비교. 조건식 2도 참이 아닐 경우 else 내 중괄호 실행
- `switch`문도 있음(자주 쓰지 않는 사람 많음 길이가 길어짐. if문을 더 선호)=> 자동완성 활용하기

##### 반복문
- `for(변수선언; 조건; 증감식) {실행할 코드}`
```dart
// 1부터 100까지 출력
 for (var i = 1; i <= 100; i++) {
    print(i);
  }
```
- `while(조건) { 실행할 코드 }`

##### 주석

## 함수
- `반환타입 함수명(매개변수) { 실행코드 }`
